# Atya.Foundation.Primitives

Low-level reusable primitives for Atya foundation packages.

## Included primitives

- `Error`
- `Result`
- `Result<TValue>`
- `ValueObject`
- `StronglyTypedId<TValue>`
- `PagedRequest`
- `PagedResult<T>`

## Installation

```bash
dotnet add package Atya.Foundation.Primitives
```
